# 💳 Cashfree Payment Setup Guide

## Does Cashfree Work Worldwide?

**Limited:**
- ✅ India - Full support (UPI, Cards, Net Banking)
- ⚠️ Other countries - Only cards, limited features

**For global payments:** Stripe is better, but Cashfree is perfect for India!

### 2. Bank Account:

## Steps to Get Cashfree Keys:

1. Go to: https://dashboard.cashfree.com
2. Sign up with email/phone
3. Complete KYC (Know Your Customer)
4. Get API Keys:
   - `CASHFREE_APP_ID`
   - `CASHFREE_SECRET_KEY`
5. Add your bank account for payouts
