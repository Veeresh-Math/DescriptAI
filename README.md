## Getting Started

### Development
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build for Production (Static Export)
```bash
npm run build
```

This creates a static export in the `out/` folder that can be deployed to:
- Cloudflare Pages (FREE)
- GitHub Pages (FREE)
- Netlify (FREE)
- Vercel (FREE)
